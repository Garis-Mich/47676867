## Описание программы 
Weather Diary (Дневник погоды) - приложение позволяющее добавлять, просматривать и фильтровать записи о погоде, сохранять данные.
## Требования для запуска 
Что нужно установить на компьютер: 
- Python 3.10 или выше
- Библиотеки: `tkinter', 'json', 'datetime'.
